# Horgász Napló Pro

GitHub Pages kompatibilis, telefonon is használható horgász napló webapp.

## Fő funkciók

- GPS koordináta lekérése telefonon
- Élő időjárás Open-Meteo API-val, bővített adatokkal
- Mentett horgászatok visszanézése mentéskori időjárással
- Vízterület, hal, csali, etetőanyag, horog és módszer kiválasztás
- Csali / etetőanyag / horog / módszer adatbázis bővíthető képpel
- Saját vízterület felvétele képpel és koordinátával
- Halfaj adatlap tilalmi idővel és méretadattal
- Statisztikák
- JSON export/import biztonsági mentéshez
- PWA alap: mobilon kezdőképernyőre kitehető

## GitHub Pages

Töltsd fel a tartalmat egy GitHub repository gyökerébe, majd Settings → Pages → Deploy from branch.

## Fontos

A GPS csak HTTPS alatt működik, ezért GitHub Pages-en jó. Az időjárás élő lekéréséhez internet kell. A tilalmi idők induló adatok, mindig ellenőrizd az aktuális országos és helyi horgászrendet.
