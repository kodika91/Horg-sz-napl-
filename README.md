# Horgász Napló

GitHub Pages-re feltölthető, backend nélküli webapp.

## Feltöltés
Az `index.html`, `app.js`, `data.js`, `style.css`, `manifest.json`, `sw.js` és az `assets` mappa legyen közvetlenül a repository gyökerében.

## Fontos
Az élő GPS és időjárás GitHub Pages HTTPS linken működik. A telefonon mentett adatok localStorage-ban vannak, export/import JSON mentéssel.
